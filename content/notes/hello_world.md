## 安装和设置新版 Quartz {#安装和设置新版-quartz}

```shell
# 1. 使用 npx 创建一个新的 Quartz 站点 (例如，叫 my-quartz)
# 这会在当前目录下创建一个叫 my-quartz 的文件夹
npx quartz create my-quartz

# 2. 进入该目录
cd my-quartz

# 3. 预览你的站点
npx quartz build --serve
```
