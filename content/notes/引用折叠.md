`引用折叠` (Reference Collapsing) 是 C++ 语言中的一个底层规则。它规定了当“引用的引用”在特定上下文（主要是模板类型推导或 `typedef` / `using` ）中出现时，它们应该如何“折叠”成一个单一的引用。


## 引用折叠规则 {#引用折叠规则}

折叠规则可以用一句话概括：只要有左值引用 `&` 参与，最终结果就是左值引用 `&` 。只有当所有参与者都是右值引用 `&&` 时，结果才是右值引用 `&&` 。

| 推导前                 | 折叠后      |
|---------------------|----------|
| T&amp; &amp;           | T&amp;      |
| T&amp; &amp;&amp;      | T&amp;      |
| T&amp;&amp; &amp;      | T&amp;      |
| T&amp;&amp; &amp;&amp; | T&amp;&amp; |


## 举例说明 {#举例说明}


### 示例1：普通模板函数 {#示例1-普通模板函数}

```cpp
template<typename T>
void func(T&& x) { }

int main() {
    int a = 10;
    func(a);    // a 是左值
    func(20);   // 20 是右值
}
```

分析：

1.  `func(a)`
    -   实参是左值 ---&gt; `T` 被推导为 `int&`
    -   替换进去： `T&&` ---&gt; `int& &&`
    -   按折叠规则 ---&gt; `int&`
    -   实际类型： `void func(int& x)`

<!--listend-->

1.  `func(20)` ：
    -   实参是右值 ---&gt; `T` 被推导为 `int`
    -   替换进去： `T&&` --&gt; `int&&`
    -   无需折叠： `void func(int&& x)`
    -   实际类新： `void func(int&& x)`

`T&&` 在模板中既能绑定左值，也能绑定右值。


### 示例2：非模板函数 {#示例2-非模板函数}

```cpp
void func(int&& x) {}

int main() {
    int a = 10;
    func(a);     // ❌ 错误，左值不能绑定到右值引用
    func(20);    // ✅ OK
}
```

> 这里的 int&amp;&amp; 不是模板参数，不触发引用折叠，所以只能接右值。


## 在 `std::forward` 中的作用 {#在-std-forward-中的作用}

`std::forward` 就是依靠引用折叠来判断该返回左值还是右值。

```cpp
template<typename T>
void wrapper(T&& arg) {
    foo(std::forward<T>(arg));
}
```

经过引用折叠后，如果 arg 是左值， `std::forward` 返回左值，如果 arg 是右值，则 `std::forward` 返回右值。

| 实参      | 推导的 T | arg 的类型 | forward 返回类型 |
|---------|-------|---------|--------------|
| 左值（int a） | `int&` | `int&`  | `int&` （折叠） |
| 右值（20） | `int`  | `int&&` | `int&&` （保持右值） |


## 直观理解图 {#直观理解图}

```cpp
         ┌───────────────┐
         │   模板参数 T   │
         └──────┬────────┘
                │
                ▼
         ┌────────────────────┐
         │  形成 T&& 类型     │
         └────────┬──────────┘
                  │
     ┌────────────┴─────────────┐
     │                          │
左值实参（T=int&）      右值实参（T=int）
     │                          │
     ▼                          ▼
 int& &&  → 折叠为 int&     int&& → 保留 int&&
```
